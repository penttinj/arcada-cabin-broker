# Arcada Cabin Broker
*IA-2-017 - Web Services and Cloud Technology*
![](https://images.unsplash.com/photo-1449158743715-0a90ebb6d2d8?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=1350&q=80)

A restful API for users to register their cabins, advertise them for renting, and manage their bookings.

## Getting started
1.  `npm install`
2.  `npm run dev`
3.  Server listens by default on port 3000  

All required configurations such as MongoDb credentials and JWT secret should be written to a `.env` file. Use `.env.sample` as template.
